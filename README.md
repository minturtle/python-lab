# python-lab
랩실 딥러닝 기반 웹 프로젝트

2가지 기능 제공
- 음성 파일을 입력받아 텍스트화 + 요약본
- 주어진 음성 파일을 기반으로 tts 만들기

## Usage

### 설치
- 아래의 명령어를 실행해 whisper 설치
```
pip install git+https://github.com/openai/whisper.git 
```
- 로컬 시스템에 ffmpeg 설치
- 아래의 명령어를 실행해 kobart 설치
```
pip install git+https://github.com/SKT-AI/KoBART#egg=kobart
```
